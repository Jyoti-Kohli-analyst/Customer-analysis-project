**Customer Segmentation analysis project**
Overview:
 This project analysis customer data using Python.

Dataset:
 File: Mall_Customers.csv

Business Question solved :
1. How many customers are in dataset?
2. What is average age of customer?
3. What is average annual income of customers?
4. Do males or females have higher spending scores?
5. Which age group has highest spending score?
6. What is the relation between income and spending score?
7. Are there any customer with high income but low spending?
8. Which customer can be considered high-value customers?

Tools used :
 Python
 Pandas
 Matplotlib
 Seaborn

Key Work Done:
 Performed data exploration and analysis
 Analyzed customer demographics and spending behavior
 Identified customer segments based on income and spending
 Created visualization for better insights

charts and visualization for insights
